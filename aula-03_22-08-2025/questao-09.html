Questão 9:

Em um sistema de aquecimento de uma usina termelétrica, o balanço de massa e energia para três fluxos de entrada, \(m_1, m_2, m_3\), é descrito por:

\[ \begin{cases} 2m_1 - m_2 + 3m_3 = 20 \\ 4m_1 + 2m_2 - m_3 = 10 \end{cases} \]

Esse sistema tem solução?

Análise do Problema

Olá! Para resolver esta questão, precisamos determinar se o sistema de equações lineares apresentado possui uma solução. Temos aqui um sistema com duas equações e três incógnitas (\(m_1, m_2, m_3\)).

Quando o número de incógnitas é maior que o número de equações, o sistema pode ter infinitas soluções ou nenhuma solução. Ele nunca terá uma solução única. Vamos verificar qual é o caso para este sistema.

Passo 1: Organizar o Sistema

A primeira etapa é organizar o sistema para facilitar a resolução. O método de eliminação é uma ótima abordagem aqui. O objetivo é eliminar uma das variáveis para encontrar uma relação entre as outras duas.

Lógica: Vamos eliminar a variável \(m_2\). Para isso, podemos multiplicar a primeira equação por 2 e somá-la à segunda equação. Isso fará com que os coeficientes de \(m_2\) se anulem.

O sistema inicial é:

\[ \begin{cases} 2m_1 - m_2 + 3m_3 = 20 \quad \text{(Equação 1)} \\ 4m_1 + 2m_2 - m_3 = 10 \quad \text{(Equação 2)} \end{cases} \]

Passo 2: Eliminação da Variável \(m_2\)

Lógica: Multiplicando a Equação 1 por 2, igualamos o coeficiente de \(m_2\) (em módulo) ao da Equação 2, mas com o sinal oposto.

Multiplicando a Equação 1 por 2:

\[2 \times (2m_1 - m_2 + 3m_3) = 2 \times 20\] \[4m_1 - 2m_2 + 6m_3 = 40 \quad \text{(Nova Equação 1)}\]

Agora, somamos a "Nova Equação 1" com a Equação 2:

\[(4m_1 - 2m_2 + 6m_3) + (4m_1 + 2m_2 - m_3) = 40 + 10\] \[(4m_1 + 4m_1) + (-2m_2 + 2m_2) + (6m_3 - m_3) = 50\] \[8m_1 + 5m_3 = 50\]

Passo 3: Expressar uma Variável em Função de Outra

Lógica: A equação resultante (\(8m_1 + 5m_3 = 50\)) nos mostra uma relação direta entre \(m_1\) e \(m_3\). Podemos isolar uma dessas variáveis para expressá-la em termos da outra. Vamos isolar \(m_1\).

A partir de \(8m_1 + 5m_3 = 50\):

\[8m_1 = 50 - 5m_3\] \[m_1 = \frac{50 - 5m_3}{8}\]

Passo 4: Encontrar a Expressão para a Terceira Variável

Lógica: Agora que temos \(m_1\) em função de \(m_3\), podemos substituir essa expressão em uma das equações originais para encontrar \(m_2\) também em função de \(m_3\). Usaremos a Equação 1, pois parece mais simples.

Substituindo \(m_1 = \frac{50 - 5m_3}{8}\) na Equação 1 (\(2m_1 - m_2 + 3m_3 = 20\)):

\[2 \left( \frac{50 - 5m_3}{8} \right) - m_2 + 3m_3 = 20\]

Simplificando a fração:

\[\frac{50 - 5m_3}{4} - m_2 + 3m_3 = 20\]

Agora, vamos isolar \(m_2\):

\[m_2 = \frac{50 - 5m_3}{4} + 3m_3 - 20\]

Para somar os termos, encontramos um denominador comum (4):

\[m_2 = \frac{50 - 5m_3 + 12m_3 - 80}{4}\] \[m_2 = \frac{7m_3 - 30}{4}\]

Conclusão

Conseguimos expressar tanto \(m_1\) quanto \(m_2\) em termos de uma terceira variável, \(m_3\). Isso significa que, para cada valor que escolhermos para \(m_3\), encontraremos um valor correspondente para \(m_1\) e \(m_2\).

As soluções do sistema são da forma:

\[ \left( m_1, m_2, m_3 \right) = \left( \frac{50 - 5m_3}{8}, \frac{7m_3 - 30}{4}, m_3 \right) \]

Como podemos atribuir infinitos valores a \(m_3\), o sistema possui infinitas soluções.

Resposta Final: Sim, o sistema tem solução. Mais especificamente, ele é um Sistema Possível e Indeterminado (SPI), o que significa que ele possui infinitas soluções.
